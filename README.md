# Vier scripts voor Gent 2040

## To deploy:

* Clone this repository
* Navigate to that directory using the terminal / command line
* Type:

``` shell
bundle exec jekyll serve
```

or

``` shell
bundle exec jekyll serve --watch
```

To build as changes are being made.
